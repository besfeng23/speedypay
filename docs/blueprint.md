# **App Name**: SpeedyPay Marketplace Console

## Core Features:

- Dashboard Overview: Display key financial metrics like total gross volume, platform fees, net remittances, pending/failed settlements, active merchants, and recent activities for at-a-glance insights.
- Merchant Management: Manage merchant profiles including onboarding status, settlement details, fee configurations, and property associations, with CRUD operations for efficient administration.
- Payment Transaction Viewer: View and filter all payment transactions, including gross amount, fees, net amount, and their detailed status timelines with reference numbers and status filters.
- Settlement & Remittance Orchestration: Automate calculation of platform fees, merchant net amounts, and generate settlement instructions for each payment with traceable statuses, powered by server-side trusted logic to ensure data integrity.
- Flexible Fee Configuration: Define and manage various fee structures (percentage, fixed) with robust server-side validation to ensure mathematical correctness, proper rounding, and prevention of impossible configurations or negative net settlements.
- Comprehensive Audit Logs: Maintain a full, immutable audit trail of all critical system events including merchant creation/updates, fee configuration changes, payment status transitions, and settlement orchestration activities for complete traceability and compliance.
- Demo Simulation Tool: Utilize an AI-powered tool to simulate the end-to-end payment lifecycle, from payment creation to fee calculation, settlement record generation, and remittance, including various success and failure scenarios for effective demonstration of the full business model.

## Style Guidelines:

- Primary interactive color: A deep, professional blue (#2662D9) to convey trust and stability, aligning with a serious enterprise tone.
- Background color: A very light, cool grey (#F0F1F5) providing a clean, spacious canvas for data, derived from a heavily desaturated variant of the primary hue.
- Accent color: A vibrant violet-blue (#6147EB) for calls to action, highlights, and important notifications, offering energetic contrast while maintaining a professional feel.
- Body and headline font: 'Inter' (sans-serif) for its modern, objective, and highly legible characteristics, ensuring readability across all data cards and tables.
- Utilize a consistent set of clean, line-based or glyph-based vector icons for navigation, data visualization, and status indicators, reinforcing the polished enterprise aesthetic.
- Implement a clean left-sidebar navigation structure with ample whitespace, modern data cards, readable tables, and strong visual badges for status elements, all optimized for a professional admin experience.
- Employ subtle and functional animations for transitions, data loading states, and interactive elements, enhancing the user experience without causing distractions or slowing efficiency in an enterprise environment.
